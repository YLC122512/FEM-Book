# 三维杆单元有限元程序

## 运行环境
- Python 3.7 或更高版本
- 依赖库：NumPy

安装依赖：
```bash
pip install numpy
程序文件
truss3d.py：主程序，包含所有函数定义及验证算例。

运行方法
在终端中进入程序所在目录，执行：

bash
python truss3d.py
功能说明
程序自动运行两个验证算例，并输出：

单元长度、方向余弦

6×6 全局刚度矩阵

给定节点位移下的应变、应力、轴力

刚度矩阵对称性、特征值（奇异性/半正定性）、刚体位移检验

刚度矩阵物理意义展示（单位位移对应的节点力）

附加题（可选）：四面体桁架的组装、求解与杆件应力输出。

输入参数（代码内直接修改）
节点坐标 x1, x2

弹性模量 E (Pa)

截面积 A (m²)

节点位移 de (m)

输出示例

错误处理
若两个节点坐标重合，程序抛出异常并提示“退化单元”。
