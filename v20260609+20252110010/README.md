# 总体刚度矩阵组装与桁架结构求解程序

## 运行环境
- Python 3.7 或更高版本
- 依赖库：NumPy

安装依赖：
```bash
pip install numpy
程序文件
main.py：主程序入口

model.py：模型数据类

element.py：单元刚度矩阵与应力计算

assembly.py：LM矩阵生成与总体刚度矩阵组装

solver.py：边界条件处理（缩减法 + 罚函数法）

postprocess.py：后处理（应力/轴力输出）

运行方法
在终端中进入程序所在目录，执行：

bash
python main.py
切换算例：修改 main.py 中的 input_file 变量（可选 input_1d.json 或 input_2d.json）。

输入文件（JSON格式）
程序通过 JSON 文件读取模型数据，示例（input_2d.json）：

json
{
  "Title": "2D two-bar truss",
  "nsd": 2,
  "ndof": 2,
  "nnp": 3,
  "nel": 2,
  "nen": 2,
  "E": [1.0, 1.0],
  "CArea": [1.0, 1.0],
  "x": [1.0, 0.0, 1.0],
  "y": [0.0, 0.0, 1.0],
  "IEN": [[1, 3], [2, 3]],
  "fixed_dof": [1, 2, 3, 4],
  "fixed_value": [0.0, 0.0, 0.0, 0.0],
  "force_dof": [5, 6],
  "force_value": [10.0, 0.0]
}
nsd：空间维度（1或2）

ndof：每节点自由度数（1或2）

nnp：节点总数

nel：单元总数

E：弹性模量数组（长度 = nel）

CArea：截面积数组（长度 = nel）

x, y：节点坐标

IEN：单元连接（1‑based 节点编号）

fixed_dof/fixed_value：位移边界条件（自由度编号 1‑based）

force_dof/force_value：节点载荷

功能说明
程序自动运行指定的算例，并输出：

总体刚度矩阵（对称性检查）

节点位移

约束反力

每个单元的长度、方向余弦、应力、轴力

刚度矩阵性质：行列式（边界处理前/后）、对角元非负、稀疏性

错误处理
若 JSON 文件不存在或格式错误，程序输出明确错误信息并退出。

若两个节点坐标重合（零长度单元），element.py 会抛出异常。

处理边界条件前检查刚度矩阵奇异性，行列式接近于 0 时提示。
