# 有限元平衡方程组求解与误差分析程序

## 运行环境
- Python 3.7 或更高版本
- 依赖库：NumPy, SciPy, Matplotlib（可选）

安装依赖：
```bash
pip install numpy scipy matplotlib
# 若需 PARDISO 求解器（可选）：
pip install pypardiso
程序文件
ldlt.py：自行实现的 LDL^T 分解与求解

solver_interface.py：统一求解接口（LDLT / 稀疏求解器）

poisson.py：Poisson 方程 Q4 单元有限元矩阵装配

utils.py：残差、条件数、计时等工具

main.py：所有验证算例执行脚本

run_all.py：一键运行所有算例

运行方法
在终端中进入程序所在目录，执行：

bash
python run_all.py
或单独运行 python main.py。

功能说明
程序自动运行以下验证算例，并输出结果：

复用 2.3 作业：一维杆结构、二维桁架结构的位移、反力、单元应力。

三对角对称正定矩阵（n=10,100,500）：验证 LDL^T 求解精度与时间。

非正定矩阵检测：测试 [[1,2],[2,1]]，程序应抛出异常。

病态矩阵误差分析：2×2 矩阵条件数 ~40002，分析残差与误差关系。

大规模 Poisson 方程：Q4 单元求解，网格 50×50 以上，调用稀疏求解器，输出最大误差、L2 误差、相对残差。

多载荷工况（可选）：展示一次分解、多次回代的效率优势。

活动列/轮廓存储：理论说明，不单独编码。

输入参数（代码内直接修改）
三对角矩阵规模 n

Poisson 方程网格密度 nx, ny

病态矩阵元素（可调整）

求解器类型（"scipy" / "pardiso"）

输出示例（部分）

错误处理
若矩阵非正定（LDL^T 分解时主元 ≤ 0），程序抛出异常并提示。

若节点坐标重合（Poisson 网格无此问题），可类似处理。

若 PARDISO 未安装，自动回退至 SciPy 求解器。
